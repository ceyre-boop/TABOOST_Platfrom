⚡ How to Use Auto Upload:

  1. Get a GitHub token (one-time setup) - click the link above
  2. Save the token somewhere safe (password manager)
  3. Drop your CSV on the box above
  4. Paste your token in the field that appears
  5. Click "Upload & Update Site"
  6. Wait 2 minutes - site updates automatically!

🔒 Your token is stored only in your browser and used only to commit files.

📥 How to Use Manual Download:

  1. Drop your CSV on the box above
  2. Click "Download data.js"
  3. Click "Open GitHub" to go to the file
  4. Click the ✏️ pencil icon to edit
  5. Select all (Ctrl+A), paste (Ctrl+V)
  6. Scroll down, click "Commit changes"
  7. Wait 1 minute for site to update
